# Unfaithful Chain-of-Thought

**Substrate:** explanatory
**SEP instance from:** [The Ærr Trinity §5.4](../../papers/aerr-trinity/the-aerr-trinity.md#54-constructive-instances)

A Surrogate Existence Postulate (SEP) instance.

| Component | Value |
|---|---|
| Demand | Faithful explanation of why an answer was produced |
| Local check | Explanation plausibility / coherence |
| Cost floor | Causal faithfulness testing (counterfactuals, interventions, mechanistic verification) |
| Surrogate signature | Fluent rationale rationalizes an answer without naming the actual causal driver |
| κ-positioning | Rate at which explanations are intervention-tested rather than accepted as fluent |

## Reference

Turpin, M., Michael, J., Perez, E., and Bowman, S. R. (2023). Language models don't always say what they think: unfaithful explanations in chain-of-thought prompting. *Advances in Neural Information Processing Systems 36*. [arXiv:2305.04388](https://arxiv.org/abs/2305.04388)

## Trinity reading

Explanation plausibility (the local gauge) can diverge arbitrarily from causal faithfulness (the demand). The Phantom Floor Threshold sits exactly between fluent rationalization and verified mechanism.
